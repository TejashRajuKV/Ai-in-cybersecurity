# CyberRakshak AI Cybersecurity Suite

CyberRakshak is an advanced cybersecurity intelligence dashboard powered by AI logic. It provides real-time monitoring and detection for various digital threats.

## 🚀 Project Structure

- **`backend/`**: Flask-based API serving AI detection models.
- **`frontend/`**: Vite + React dashboard with premium design.
- **`docs/`**: Documentation and architecture guides.

## 🛠️ Tech Stack

- **Backend**: Python 3.x, Flask, Scikit-learn, Pandas.
- **Frontend**: React 18, Vite, Framer Motion, Lucide Icons.
- **AI/ML**: NLP for Scam/News detection, Behavioral anomaly detection.

## 📦 Getting Started

### Backend Setup
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python app.py`

### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`
